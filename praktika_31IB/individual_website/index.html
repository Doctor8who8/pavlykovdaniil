Теоретическая база

  • HTML: Язык гипертекстовой разметки, основанный на использовании тегов и атрибутов для структурирования документа.
  • CSS: Каскадные таблицы стилей — технология для представления Web-страниц, определяющая цвета, шрифты, выравнивание и оформление.
  • Формы: Элементы, используемые для отправки данных на сервер и получения данных с сервера (например, элементы <input>, <textarea>, <select>).
  • JavaScript: Язык сценариев, где переменные могут хранить значения различных типов: строки, числа, булевы значения и объекты.
  • Bootstrap: Бесплатный набор инструментов (фреймворк) с открытым исходным кодом, созданный для быстрой вёрстки сайтов и веб-приложений
